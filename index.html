Popular Courses

Basic STEM

$20

Basic STEM

This program has three levels. Young people enter the world of robotics, programming and building robot models for the future in the field of physics and space. Participants will install models of satellite, spacecraft, robot arm, etc. to apply what they have learned about programming in a fun way. Kids study a new set of topics at each level.

Scratch Gamer

$30

Scratch Gamer

Scratch is your gateway to the world of drag-and-drop programming Language to understand programming concepts by making great games. Each game includes easy-to-follow instructions that help you learn fast. Would you like to be a game maker that is played by millions of people around the world? In this program, you will design many games such as: Maze Runner-- Asteroid Breaker- Snake- Fruit Slicer.

Photoshop

$30

Photoshop

The most popular graphics program for all design workers is Photoshop. Students understand all the basics of the program as well as the relationship between the camera and art and design elements in the image and the modification of images and colors and remove defects. Students learn through real projects that encourage participants to enter the world of professional graphic designers..

Mobile app

Mobile app

Do you want to make a mobile application for any existing idea? In this main program, we use the MIT App Inventor to design and program Android phone applications using the Visual Programing Environment and run the application on smartphones and tablets. Start your future now!.
